# Awesome Expo Starter

Minimal boilerplate code for a full-stack cross-platform app with self-hosted authentication and database out of the box.

## The stack
- Expo : For web and mobile frontend development
- Hono : The fastest node backend
- DrizzleORM : Effective PostgreSQL schemas, connections and migrations
- Better Auth : Reliable self-hosted auth with all features, google OAuth already setup

## Required Environment Variables

PORT
POSTGRES_USER
POSTGRES_PASSWORD
DATABASE_URL
BETTER_AUTH_SECRET
GOOGLE_CLIENT_ID
GOOGLE_CLIENT_SECRET
FRONTEND_URL
